\section{Schedule}
\label{section:schedule}

The Prometheus project is structured into a 16-week development lifecycle. The schedule is divided into five distinct operational sections that progress from foundational infrastructure to final system evaluation, ensuring that the Computer Vision (CV) pipeline is validated before full integration with the user interfaces.

The timeline prioritizes the development of the core AI models and the Edge-to-Cloud metadata pipeline during the first half of the project. This is followed by parallel development of the iOS mobile application for members and the analytical Web Dashboard for facility owners. The final weeks are dedicated to rigorous system-wide evaluation to verify that the CV accuracy meets the established success criteria.

\subsection{Phase Descriptions}
Based on the visual timeline presented in the Gantt chart, the project phases are defined as follows:

\begin{itemize}
    \item \textbf{April 2026: Phase 1 — Infrastructure Setup, Data Acquisition, and Prototyping:} This initial phase focuses on the physical installation of the CCTV hardware and site surveys at the facility. Key tasks include data collection and annotation, establishing basic human-to-machine classification models, and developing early prototypes.
    \item \textbf{January 2027: Phase 2 — AI Model and Backend Pipeline Development:} Resuming after a scheduled break, this phase involves rigorous data preprocessing and the training of the occupancy detection models. The focus is on model validation, optimization, and the establishment of the core CV-to-Database pipeline to handle metadata transmission.    
    \item \textbf{February 2027: Phase 3 — Mobile Application and Web Dashboard Implementation:} This phase centers on the development of the user interfaces for both gym members and owners. Work includes building the iOS application (incorporating the machine tracker and notification system) and the web-based dashboard for spatial heatmaps, followed by comprehensive UI/UX refinement and testing.
    \item \textbf{March 2027: Phase 4 — System Integration and Final Evaluation:} The concluding phase involves system-wide API wiring and integration to ensure all components communicate seamlessly. Efforts are dedicated to accuracy and performance testing against real-world scenarios, culminating in the production of the final evaluation report and project submission.
\end{itemize}
